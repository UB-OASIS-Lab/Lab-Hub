# Publications

This folder provides an index of peer-reviewed publications, reports,
and preprints produced by the lab.

Each publication is documented using a **single Markdown file** that includes:
- full citation
- abstract
- links to code, data, and models
- DOI or preprint link

This repository does **not** host full manuscripts or reviewer-only materials.

---

## How Publications Are Organized
- One Markdown file per paper
- Filename format:
  YYYY_short-title.md

Example:
2025_energy_poverty_grid_resilience.md

---

## For Lab Members
- Create a publication entry when a manuscript is submitted or posted as a preprint.
- Update the entry with DOI and journal information upon acceptance.
- Ensure links to project repositories are correct and public.
